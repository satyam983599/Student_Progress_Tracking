# 🎓 School ERP & Student Performance Analytics System

A full-stack School ERP and Student Performance Analytics System built using the MERN Stack. This application helps school administrators manage students, subjects, examinations, marks, and performance reports through a modern and user-friendly dashboard.

## 🚀 Features

### 👨‍🎓 Student Management

* Add, update, view, and delete student records
* Filter students by class and section
* Roll number-based student search
* Student profile and academic information management

### 📚 Subject Management

* Add new subjects dynamically
* Delete existing subjects
* Subject code management
* Automatic subject synchronization with Marks Entry module

### 📝 Marks Entry System

* Exam-wise marks entry
* Multiple examination support:

  * Unit Test 1
  * Unit Test 2
  * Project 1
  * Half Yearly
  * Unit Test 3
  * Unit Test 4
  * Project 2
  * Final Exam
* Duplicate subject validation
* Teacher remarks support

### 📊 Reports & Analytics

* Student report card generation
* Overall percentage calculation
* Subject-wise performance analysis
* Exam-wise marks breakdown
* Pass/Fail status calculation
* Printable report cards
* PDF report generation

### 🔐 Authentication & Security

* Admin Registration
* Admin Login
* JWT Authentication
* Protected Routes
* Secure password hashing using bcryptjs

### 🎨 Modern Dashboard

* Responsive UI
* Sidebar navigation
* Professional admin dashboard
* Mobile-friendly design
* Modern Tailwind CSS components
